# 3D-Graphics-Project
CS 4361 Graphics Project

The final project has been compiled under the "3D Graphics Project"
folder. Open "Game.app" to run the game.

If you have the Unity Game Engine installed on your machine, you
Can open the 3D Graphics Project workspace. Under File > Build and Run,
You can run the game.
